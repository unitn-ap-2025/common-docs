Enum BasicResource 

Source 
pub enum BasicResource {
    Oxygen(Oxygen),
    Hydrogen(Hydrogen),
    Carbon(Carbon),
    Silicon(Silicon),
}
Expand description

An enum that provides a unified type for all possible basic resources.

This enum wraps every generated basic resource struct (e.g., Oxygen, Hydrogen) into a single type. It is useful when you need to store or pass around any basic resource without knowing its specific concrete type at compile time.
